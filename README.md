# Pulse Multi-Monitor
A centralized Laravel-based dashboard for monitoring multiple Laravel apps using Laravel Pulse.

## Features
- Collect metrics from multiple Laravel apps.
- Visualize CPU, memory, and disk usage.
- Set up alerts and notifications.

## Installation
1. Clone the repo: `git clone https://github.com/Shahirul22/laravel-pulse-multiple.git`
2. Run `composer install`.
3. Set up `.env` file and migrate: `php artisan migrate`.
